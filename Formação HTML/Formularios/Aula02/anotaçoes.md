O <input type="text"> é um campo de caracteres

O <input type="number"> é um campo exclusivo para numeros. E podendo definir o min e o max com os atributos "min" e "max": <input type="number" min="0" max="99" step="5">. O atributo step é o que define que vai pular de tanto em tanto.

O <input type="button"> não é semanticamente correto, o mais usado é a tag <button></button>

O <input type="range"> forma um range na pagina com vc podendo tbm definir o max e min, atraves dos atributos "min" e "max"

O <input type="email"> é o navegador verificando se é email que está sendo inserido

O <input type="url"> é bem parecido porem ele pede uma url

O <input type="date"> é para salvar as datas

O <input type="week"> é para vc escolher a semana de determinado ano

O <input type="month"> é para selecionar o mes de determinado ano

O <input type="checkbox"> é para abrir uma caixa que pode ou não ser selecionada

O <input type="radio"> é para definir apenas uma caixinha de duas.

O <input type="hidden"> fica invisivel para o navegador.

O <input type="file" multiple> é para enviar arquivos, e com o atributo multiple, é para enviar mais de um arquivo.

O <input type="search"> é um mecanismo de busca.
